## Notebooks

### Notebook 1 — Dataset Analysis
Exploratory analysis of the MER-TRANS 2026 trial, test, and gold datasets for Spanish and Italian. Computes descriptive statistics (token length, number of lines, simplification ratio) and generates visualisations of the data distributions.

### Notebook 2 — Augmented Development Set
Constructs the augmented Spanish development set used for internal evaluation before the official submission. Starting from the 19 base segments (ES trial + CA→ES + IT→ES translations), applies three rounds of back-translation via different language pivot chains to produce 76 diverse segments.

### Notebook 3a — Inference: SZS, SFS, AZS (4 blocks)
Runs inference with the three lightest prompt configurations — Simple Zero-Shot (SZS), Simple Few-Shot (SFS), and Advanced Zero-Shot (AZS) — on a 346-segment dataset. Supports Spanish test set, Italian test set, and FEINA sample. Segments are processed in 4 blocks of ~87 to manage the Groq free-tier daily token limit.

### Notebook 3b — Inference: AFS (6 blocks)
Runs inference with the Advanced Few-Shot (AFS) configuration, which uses a longer prompt and therefore requires 6 blocks of ~58 segments. Supports Spanish test set, Italian test set, and FEINA sample.

### Notebook 3c — Inference: Augmented Development Set
Runs all four prompt configurations (SZS, SFS, AZS, AFS) on the 76-segment augmented development set in a single run, without block splitting. Outputs one CSV per configuration for evaluation.

### Notebook 4 — FEINA Sample Preparation
Selects a reproducible random sample of 346 segments from the FEINA corpus using a fixed random seed (42).

## System

The HULAT1 system is based on **Llama 3.3 70B** accessed via the Groq API. Four prompt configurations were evaluated:

| ID | Name | Description |
|---|---|---|
| SZS | Simple Zero-Shot | Basic simplification prompt, no examples |
| SFS | Simple Few-Shot | Basic prompt with 3 in-context examples |
| AZS | Advanced Zero-Shot | Prompt enriched with UNE 153101:2018 EX guidelines, no examples |
| AFS | Advanced Few-Shot | UNE-based prompt with 3 in-context examples |
